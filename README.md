## Getting Started

### Step 1: Install Python
Download from https://www.python.org/downloads/ and check "Add Python to PATH".

### Step 2: Run the Setup Script
```
python setup.py
```
Installs: `torch`, `datasets`, `scikit-learn`, `matplotlib`, `seaborn`, `numpy`, `pandas`, `notebook`

### Step 3: Launch Jupyter
```
jupyter notebook
```
Then open `imdb_transformer_project.ipynb`.

### Step 4: Run the Notebook
Click **Kernel → Restart & Run All**

> **Note:** Training takes ~5 minutes on a GPU, ~60-90 minutes on CPU.
> A CUDA-capable GPU is strongly recommended for the training loop.

---

## Troubleshooting

**No GPU / training is slow?**
Use Google Colab (free GPU):
1. Upload the `.ipynb` to https://colab.research.google.com
2. Set Runtime → Change runtime type → GPU
3. Run all cells

**"ModuleNotFoundError"?**
```
pip install torch datasets scikit-learn matplotlib seaborn notebook
```

**PyTorch install fails?**
Visit https://pytorch.org/get-started/locally/ for your specific OS + CUDA version.
